A simple and lightweight integration plugin with free Cookie.gg GDPR banner-as-a-service platform.

== Description ==

Blazing fast, simple and flexible cookie consent banners for your website. GDPR and EU Cookie Law friendly. It only takes a minute to install on absolutely any dynamic or static website.

**This WordPress plugin requires a free [Cookie.gg](https://cookie.gg) account!**

= Features =

* Works on high-traffic websites with zero impact on performance
* Secure, backed by a [$10k bounty/warranty](https://cookie.gg/bounty/)
* Compatible with caching plugins, proxies and application firewalls
* Easy to integrate in WordPress or any other platform - just one line of HTML code
* Highly customizable, unlimited usage
* Trusted by hundreds of webmasters for 2+ years

== Installation ==

You will **require a banner ID** that is generated over at [Cookie.gg](https://cookie.gg).

Installing via the WordPress dashboard:

1. Browse to Plugins - Add New in your WordPress dashboard.
1. Search for "cookie.gg" using the search box on the right.
1. Find the Cookie.gg plugin in the search results and click Install Now.

Installing via FTP:

1. Upload the plugin files to the `/wp-content/plugins/pageviews` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Enjoy.

Installing via WP-CLI:

`wp plugin install cookie-gg --activate`

Installing via Git:

`cd wp-content/plugins
git clone https://github.com/pressjitsu/cookie-gg.git`

Congifiguration:

1. Get your banner ID over at [Cookie.gg](https://cookie.gg)
1. Navigate to Settings / Cookie.gg GDPR
1. Insert your 40-character banner ID from step 1 and save your settings
